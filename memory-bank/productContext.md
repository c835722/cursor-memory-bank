# Product Context

## Project Purpose
The Lotus Doors Customer Portal provides a centralized web interface for accessing various AI-powered tools, resources, and information. It serves as a platform for client education on AI technologies and provides access to automation tools and chatbots.

## Target Users
- Lotus Doors (LD): Primary customer
- Natural Velocity (NV): Solution provider
- Lotus Doors customers: End users

## User Needs
- Access to blueprint and roadmap information
- Education on AI technologies relevant to implemented solutions
- Guidelines for using the portal and its features
- Access to automation tools
- Interaction with chatbots for assistance

## Business Goals
- Provide a centralized access point for Lotus Doors to interact with AI solutions
- Educate clients on AI technologies to improve adoption and understanding
- Showcase Natural Velocity's capabilities and solutions
- Improve customer engagement and satisfaction
- Streamline access to automation tools and resources
