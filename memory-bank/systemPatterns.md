# System Patterns

## Architecture Overview
The Lotus Doors Customer Portal follows a web-based architecture with the following key components:
- Web interface built using modern web technologies
- Containerized deployment via docker-compose
- Component-based structure for modularity and maintainability
- Secure authentication and access control

## Component Structure
- Blueprint Section: Displays roadmap and strategic planning information
- Education Section: Contains AI educational content including YouTube videos from IBM Technology
- Guidelines Section: Provides usage instructions and best practices
- Automations Section: Offers access to automation tools
- ChatBots Section: Provides interfaces to AI-powered chatbots

## Design Patterns
- Component-based architecture for modularity
- Responsive design for cross-device compatibility
- Container-based deployment for portability
- Secure access controls for data protection

## Technical Decisions
- Deployable via docker-compose for easy setup
- Support for both cloud and hybrid environments
- Security best practices implementation
- Maintainable and scalable architecture
- NV color scheme for UI consistency
